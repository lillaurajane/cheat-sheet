CSS Colour Tips

Adding Colour To Your Webpage

You can add basic colour to your webpage and spice up your HTML document using CSS. Below are some of the basic properties you can use to add colour.

Property Name Description
color Foreground Colour Change the foreground colour of any element
background-color Background Colour Change the background colour of any element

Using Colours In CSS

Code Name Description
#000000 Hexidecimal Hexidecimal colours have a hash followed by 3 to 6 values
They have a 16 digit number system which includes a mix of both numbers and letters which goes beyond just using 0-9. They can use both upper and lowercase letters.
rgb(0,0,0) rgb rgb stands for Red, Green, Blue
The three values represent a colour component and can be a value of 0 to 255
hsl(0, 1%, 0%) hsl Stands for hue, saturation and lightness
The first number represents the degree of the hue and can be between 0 and 360. The second and third numbers are percentages representing saturation and lightness.
